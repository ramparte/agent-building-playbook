# wiki/ — Human-Browsable Rendered Playbook

This directory holds the **human-facing** static HTML wiki for the
Agent-Building Playbook. It is a **generated artifact**, rendered from the
canonical pattern source.

- **Source of truth:** `../patterns/` (one markdown file per pattern). Edit
  patterns there, never here.
- **Agent-facing view:** `../INDEX.md` (separate, dense, generated by
  `scripts/build-index.sh`). This wiki is the *human* counterpart.
- **What's generated here:** `index.html` (landing page grouping patterns by
  dimension tag) and `patterns/{kebab-title}.html` (one page per pattern).

## Regenerate

Run the publish step (re-renders everything from `../patterns/`):

```sh
bash .wiki/scripts/publish.sh
# or, in an agent session:  /wiki-publish
```

Then open `wiki/index.html` in a browser, or deploy `wiki/` to GitHub Pages.

> The `.html` build output is regenerable and git-ignored by default — only this
> README is tracked. To serve via GitHub Pages from this folder, either
> force-add the rendered files (`git add -f wiki/`) or render in CI on publish.
